📋 Format Template File

Pastikan file CSV/Excel Anda memiliki kolom-kolom berikut (urutan bebas, nama header harus persis seperti di bawah):

Nama Kolom (Header) Keterangan Status Contoh Nilai

Author(s) Nama penulis, pisahkan dengan koma jika lebih dari satu Wajib Andrea Hirata
ISBN Code Kode ISBN unik buku (ISBN-10 atau ISBN-13) Opsional 9789792231991
Length Jumlah halaman (angka bulat positif) Opsional 529
Language Bahasa buku. Kosong = otomatis "Indonesian" Opsional Indonesian
Total Copies Jumlah stok buku yang tersedia (angka bulat positif) Wajib 3
Shelf Location Lokasi rak penyimpanan di perpustakaan Opsional Rak A-2
Image(URL) URL gambar sampul buku (harus diawali https://) Opsional https://example.com/cover.jpg
Category Kategori atau genre buku Opsional Fiksi, Novel
